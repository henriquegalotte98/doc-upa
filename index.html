Como usar

O card informa o código e também todos os comandos de Git que o integrante deve executar.

1. Clique na SprintSelecione a aula atual.
2. Escolha seu DEVAbra somente sua responsabilidade.
3. Prepare a branchCopie os comandos do bloco “Git desta tarefa”.
4. Copie os arquivosCrie cada caminho e cole o código integral.
5. Commit e pushUse a mensagem de commit já fornecida.
6. Abra o Pull RequestEnvie para a branch de destino indicada no card.
